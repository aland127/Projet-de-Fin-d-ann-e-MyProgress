# MyProgress — Application de suivi sportif

Projet de fin d’année de L3 Informatique : application web permettant de suivre ses séances de sport, ses performances et ses objectifs.

## Description

MyProgress est une application web de suivi sportif développée en PHP / MySQL / JavaScript.  
Elle permet à un utilisateur de :

- enregistrer ses séances (musculation, cardio, vélo, natation…)
- consulter l’historique de ses entraînements
- définir des objectifs personnalisés
- visualiser automatiquement la progression en fonction des séances réalisées

Ce projet a été réalisé dans un cadre universitaire afin de mettre en pratique le développement web full-stack.

## Fonctionnalités principales

- Authentification utilisateur (inscription / connexion)
- Ajout de séances avec champs dynamiques selon le type d’activité
- Gestion des exercices (zones musculaires, poids, répétitions, distance, vitesse…)
- Historique des séances avec affichage détaillé
- Système d’objectifs avec barre de progression dynamique
- Modification et suppression des séances et des objectifs

## Technologies utilisées

- **Frontend** : HTML, CSS, JavaScript  
- **Backend** : PHP  
- **Base de données** : MySQL
